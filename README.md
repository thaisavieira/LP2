# 📚 Linguagens de Programação II – Atividades e Certificados

Repositório destinado ao armazenamento das atividades desenvolvidas e dos certificados obtidos em atividades extracurriculares da disciplina de **Linguagens de Programação II**, ministrada pela professora **Denilse Veloso**, no curso de **Análise e Desenvolvimento de Sistemas – Fatec Sorocaba**.

---

## Objetivo

Este repositório tem como finalidade:

-  Organizar as atividades práticas realizadas na disciplina;  
-  Armazenar certificados de participação em atividades extracurriculares;  
-  Documentar o progresso acadêmico;
-  Servir como registro de aprendizado;

---

## Estrutura do Repositório
- **atividades/** → Exercícios e projetos desenvolvidos na disciplina  
- **certificados/** → Certificados de participação em atividades complementares
